# Contributing to MinimalWorker

Thanks for your interest in contributing! This project is open to contributions of all kinds.

## Ways to Contribute

- **Report bugs** - Open an issue describing the problem
- **Suggest features** - Open an issue with your idea
- **Submit PRs** - Bug fixes, features, docs, tests - all welcome
- **Improve docs** - Typos, clarifications, examples

## Getting Started

1. Fork and clone the repo
2. Run `dotnet build` to verify everything compiles
3. Run `dotnet test` to make sure tests pass
4. Make your changes
5. Submit a PR

## Development

```bash
# Build
dotnet build

# Run tests
dotnet test

# Run a sample
dotnet run --project samples/MinimalWorker.Console.Sample
```

## Pull Requests

- Keep changes focused - one feature or fix per PR
- Add tests for new functionality
- Update docs if needed
- PRs are reviewed and merged when ready

## Questions?

Open an issue or start a discussion. Happy to help!
